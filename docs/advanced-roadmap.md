# Advanced Roadmap

This document tracks potential future improvements for the token-saving and context-engineering capabilities.

## Planned Enhancements

- Automated context budget tracking per session turn
- Integration with OpenCode's built-in token counting
- Skill subset preloading for common task types
- Compressed session export for debugging

## Contribution Ideas

- Add per-skill token cost estimates to skill metadata
- Create a session replay utility from handoff summaries
- Add validation for handoff completeness
- Build a skill dependency graph for automatic loading
